What is it?🌸
Soukoun Coffee Shop is a modern and welcoming online coffee shop experience designed especially for a cozy and elegant lifestyle. It allows users to explore a variety of drinks and menu items, discover the atmosphere of the coffee shop, and learn more about the activities and experiences offered inside. Through the platform, users can create an account and log in to access personalized features such as booking visits, reserving spots, and placing orders for their favorite drinks. The system is built to make ordering simple and smooth, letting customers browse the menu, add items to their cart, and complete their order with ease. Overall, Soukoun Coffee Shop combines a relaxing café vibe with convenient digital features like booking, ordering, and user accounts to create a complete and enjoyable coffee shop experience online.

✨ Features
- Browse coffee shop menu (drinks, desserts, etc.)
- Add and remove items from cart
- View cart quantity updates in real-time
- Simple checkout flow (UI only or future backend integration)
- Clean and aesthetic UI designed for a cozy coffee shop vibe

⚡ Built with React + TypeScript for performance and scalability

🛠️ Tech Stack
Frontend: React
Language: TypeScript
Styling: CSS / Tailwind 
State Management: Context API (CartContext)

📂 Project Structure
src/
│
├── components/        # Reusable UI components (cards, navbar, etc.)
├── pages/             # Main pages (Home, Cart, Menu)
├── context/           # CartContext for global state
├── assets/            # Images and icons         
└── App.tsx            # Main app entry
